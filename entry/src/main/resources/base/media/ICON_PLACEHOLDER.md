# 图标占位符

## 说明
此文件为图标占位符。在实际开发中，请替换为真实的应用图标。

## 要求
- **icon.png**: 模块图标，建议尺寸 192x192 像素
- **app_icon.png**: 应用图标，建议尺寸 192x192 像素

## 替换方法
1. 删除此占位符文件
2. 将准备好的 PNG 图标文件复制到对应目录
   - `entry/src/main/resources/base/media/icon.png`
   - `AppScope/resources/base/media/app_icon.png`

## 图标设计建议
- 使用简洁的图形，避免过于复杂
- 确保在小尺寸下仍然清晰可辨
- 遵循 HarmonyOS 设计规范
- 提供不同分辨率的图标以适配不同设备
